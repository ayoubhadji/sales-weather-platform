# Sales Weather Prediction Platform - Frontend

## 📌 Overview

The frontend is developed using **React**, **TypeScript**, and **Vite**.

It communicates with the NestJS backend using Axios.

The application provides interfaces for:

- Dashboard
- Products
- Weather
- Sales Tickets
- Promotions
- Alerts
- Sales Predictions

---

## 🛠 Technologies

- React
- TypeScript
- Vite
- React Router DOM
- Axios

## ⚙️ Installation

Go to the frontend folder

```bash
cd frontend
```

Install dependencies

```bash
npm install
```

---

## ▶️ Run the Project

```bash
npm run dev
```

Application URL

```
http://localhost:5173
```

---

## 🔗 Backend Connection

Axios is configured inside

```
src/services/api.ts
```

Default backend URL

```typescript
http://localhost:3000
```

---
